The header should remain fixed at the top of the screen while you scroll. This creates a persistent navigation that users can access from any point on the page. The hamburger menu on mobile should collapse/expand smoothly with animation effects.

The navigation menu features smooth transitions, hover states with color changes, and an active state indicator. On mobile, the menu slides down from the header with a fade-in effect. The hamburger icon transforms into an X when the menu is open.

All components follow the 2ndSTREET design system using CSS variables for colors, spacing, and transitions. This makes it easy to maintain consistency and update styles globally without modifying individual elements.
